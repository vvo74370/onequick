## 主要功能拆解

### 1. 屏幕边缘操作（Hot Corners）

屏幕的四条边、四个角，每一块都能绑上自定义动作。

默认配置下，左上角滚轮调音量，上边缘切标签页，右上角换歌——这三个动作，真的是每天都在用。

对于喜欢全屏打游戏的朋友，这个功能更实用：再也不用为了调个音量退出游戏了。

### 2. 复制后聚合搜索

选中一段文字，连按两次 Ctrl+C，再按绑定的热键，就能直接拉起一个或多个搜索引擎。

百度、Google、知乎、豆瓣……你想配哪个配哪个，专业版支持无限条，免费版只能设 5 条。

### 3. 短语替换输入

这个功能一旦配好，基本就回不去了。

比如输入 `@@email` 自动展开成你的完整邮箱，输入 `//date` 变成今天日期——常用的地址、账号、签名，统统一键搞定。专业版无限制，免费版上限 10 条。

### 4. 多步热键

自定义按键组合 + 触发条件 + 执行动作，三位一体。

你可以让某个热键只在 VS Code 里生效，触发后自动置顶窗口、运行命令、输出文字，甚至锁屏、关显示器……免费版只能设 1 条，专业版无限。

### 5. 显示器配置一键切换

多屏用户的福音。可以记录每套显示器的分辨率、刷新率、横竖屏、排列位置等，下次一个热键直接切换，告别手动调来调去的痛苦。

### 6. 其他实用小工具

屏幕按键显示（录屏教程必备）、查看应用启动命令、强制保持系统唤醒、一键关闭显示器……这些零散但高频的需求，OneQuick 都顺手打包进来了。

## 适合谁用？

说白了，OneQuick 10 是为那种**觉得现在 Windows 操作还差点意思**的人准备的。

- 程序员、设计师：需要频繁切窗口、改键映射、运行命令

- 内容创作者：短语替换能省下大把重复输入的时间
- 普通上班族：就想把调音量这件事变得稍微爽一点

它不是那种装上就要研究半天的复杂工具，大多数功能上手即用，学习曲线几乎是平的。
